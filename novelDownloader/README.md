### 使用说明

#### 在目录页或是章节页使用。

#### 按“Shift+D”来显示下载选项。

#### 关于【特定下载某些章节】功能的一些提示：可以使用带有【Snap Links】字样的附加组件或扩展或UC脚本

#### 版本命名规则

如**1.0.13**分成两段，1.0与13

1.0表示脚本功能为1.0版本。

13表示支持13个网站。
