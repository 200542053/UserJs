### 缺陷

1. 内存可能不断增加
2. 特殊技巧：龙吼(FUS RO DAH)因缺少id而不能使用。
3. 竞技场：因笔者等级不够，无法自动刷Lv300的竞技场。
4. 页面停留足够时间长后，触发警报或刷新，此时脚本恢复正常工作时，无法停止警报或刷新。
5. 待补充
6. 欢迎提供意见建议
